# **App Name**: ClassMaster

## Core Features:

- Faculty Timetable View: Displays faculty's read-only timetable with class details and material upload options.
- Real-time Notifications: Sends instant notifications to faculty for class assignments, changes, and admin approvals via web interface.
- Admin Timetable Management: Enables admins to create, manage, and assign classes, subjects, faculties, and classrooms within an interactive calendar view.
- Optimized Timetable Generation tool: Uses AI to generate multiple timetable options, checking for clashes, faculty availability, and room capacity based on constraints such as faculty leave and workload, with admin review.
- Admin Notification Dispatch: Send in-app notification to faculty.
- Faculty Profile Management: Allow faculty to view their student profiles

## Style Guidelines:

- Primary color: Deep blue (#3F51B5) for a professional and trustworthy feel.
- Background color: Light gray (#F0F2F5), nearly white, for a clean and open interface.
- Accent color: Purple (#7E57C2) for interactive elements and highlights.
- Body and headline font: 'PT Sans', a versatile sans-serif suitable for both headers and body text.
- Use clear, professional icons from a consistent set to represent different functionalities and subjects.
- Employ a clean, structured layout with a clear hierarchy and intuitive navigation.
- Subtle transitions and animations for interactive elements to improve usability without being distracting.
